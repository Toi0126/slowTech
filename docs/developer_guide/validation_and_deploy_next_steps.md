# 実装検証・デプロイ作業（別PCでの再開手順）

## 1. フロント（MVPスタブ）の動作確認

最優先で「スマホで画面遷移と入力チェックが動く」ことを確認する。

1. `web/` で簡易HTTPサーバを起動
	- 例: `cd web` → `python -m http.server 8000`
2. PCのブラウザで確認
	- `http://localhost:8000/index.html`
3. スマホ実機で確認（同一Wi-Fi）
	- PCのIPアドレスを調べ、`http://<PCのIP>:8000/index.html` を開く
4. 受け入れ条件チェック（MVP）
	- ファイル or テキスト送信で処理中（ダミー）へ遷移
	- 一定時間後に結果（ダミー）へ遷移
	- 制約超過（テキスト1万字超/300MB超/許可されない形式/2分超）でエラー表示
