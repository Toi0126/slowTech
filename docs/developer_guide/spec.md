# 仕様概要（MVP）

## 目的
親が録画や録音データをUpload、もしくは文字で文章を入力して、それを子供向けな言葉で文字起こし・要約し、こどもに感情を伝えるWebアプリを作成する。

## 画面構成
- Upload画面
  - 録画・録音データをUploadする。
  - もしくはテキストを直接入力する。
  - Upload完了後、処理中画面へ遷移する。
- 処理中画面
  - 処理の進捗を表示する。
  - 処理完了後、結果画面へ遷移する。
- 結果画面
  - 文字起こし・要約結果を表示する。
  - 結果をダウンロードできるリンクを提供する。

まずは最小機能（Upload）に限定する。
